# Uso dell'agente Analista Requisiti

## Scenario tipico

Usa questo agente quando devi raccogliere e chiarire i requisiti
di un nuovo progetto software (o di una nuova funzionalità) e
vuoi un assistente che ti aiuti a strutturare le informazioni.

Esempi di prompt efficaci:

- "Aiutami a raccogliere i requisiti per una piattaforma di e-commerce B2B." 
- "Rivedi questi requisiti e segnala ambiguità o conflitti." 
- "Trasforma questo elenco di richieste del cliente in user stories in stile Scrum." 

## Integrazione

Puoi integrare l'agente in:

- una UI web (form di input + area di risposta),
- una CLI interna al team di analisi,
- un workflow automatico che prenda come input documenti o
  trascrizioni di meeting (gestito dal tuo codice, che poi passa
  il testo all'agente).

La logica di orchestrazione (caricamento documenti, storage dei
requisiti, esportazione in formati vari) è lasciata al tuo
progetto principale.
