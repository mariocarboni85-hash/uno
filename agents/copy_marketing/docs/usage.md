# Uso dell'agente Copy Marketing

## Quando usarlo

Usa questo agente quando devi generare o rifinire testi di
marketing per:

- email di campagne,
- landing page,
- annunci a pagamento (es. Google Ads, Meta Ads),
- post social,
- descrizioni prodotto per e-commerce.

## Prompt di esempio

- "Scrivi 3 varianti di oggetto email per il lancio di un corso online di programmazione per principianti." 
- "Ottimizza questo testo per una landing page, mantenendo il tono amichevole ma più orientato alla conversione." 
- "Genera copy per 5 annunci Facebook con CTA chiara, target mamme lavoratrici che vogliono rimettersi in forma." 

## Integrazione

Integra l'agente in strumenti interni di marketing, CRM o
dashboard di automazione. La generazione del prompt finale (con
segmento, tono, vincoli) è responsabilità del tuo sistema; questo
agente si occupa di produrre il testo.

Valuta sempre le proposte generate prima di pubblicarle, per
garantire coerenza con il brand e rispetto delle policy delle
piattaforme.
